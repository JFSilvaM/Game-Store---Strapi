# 🗄️ Game Store — Strapi (Backend)

Servidor **backend** desarrollado con **Strapi**, encargado de gestionar el contenido y la API de la aplicación **Games Store**.

Este proyecto actúa como **CMS Headless** y proveedor de datos para el front‑end desarrollado en Next.js.

## 🌐 Front‑end relacionado

La aplicación cliente que consume esta API se encuentra desplegada aquí:

🔗 https://gaming-store-next.netlify.app/

Repositorio del front‑end:  
🔗 https://github.com/JFSilvaM/Games-Store---Nextjs

## 📖 Descripción

**Game Store — Strapi** proporciona una API REST para la gestión de:

- Videojuegos
- Categorías
- Imágenes y recursos
- Contenido dinámico

El objetivo del proyecto es aprender a crear y consumir una API real usando **Strapi**, separando claramente el backend del frontend.
